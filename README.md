# 📘 GradeWise - Smart CGPA Tracker

GradeWise is a modern student academic performance tracking web application designed to help students calculate, track, and analyze their CGPA across semesters in a simple and interactive dashboard.

---

## 🚀 Features

- 📊 Overall CGPA calculation across 8 semesters
- 📚 Semester-wise subject management
- ✏️ Add, edit, and delete subjects
- 💾 Data stored using LocalStorage (persistent data)
- 📈 Semester-wise CGPA table view
- 🖥️ Clean and responsive dashboard UI
- 🎯 No login required, instant usage

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Modern UI, Light Pastel Theme)
- JavaScript (Vanilla JS)
- LocalStorage (for data persistence)

---

## 📷 Project Highlights

- Dashboard-style UI
